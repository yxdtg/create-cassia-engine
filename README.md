# Create Cassia Engine

### npm

```bash
npm install -g create-cassia-engine

create-cassia-engine <name>
```

### yarn

```bash
yarn global add create-cassia-engine

create-cassia-engine <name>
```

### pnpm

```bash
pnpm add -g create-cassia-engine

create-cassia-engine <name>
```
